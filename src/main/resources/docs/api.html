1. 消息推送接口

1.1. 消息推送

URL: http://{{host}}:{{port}}/message/push

Type: POST

Author: dingpeihua

Content-Type: application/json

Description: 消息推送

Body-parameters:

ParameterTypeRequiredDescription

projectName

string

true

固定名项目名称，如：Zaful、Dresslily、Rosegal

projectPlatform

string

true

项目名称加平台，如：【Zaful-安卓APP】、【Zaful-iOS APP】

businessName

string

true

业务名称，如：【Zaful-安卓APP-下单】

behavior

string

false

行为，如：下单行为

time

string

false

时间，如:60分钟

plat

string

true

检查操作，如：下单操作

sendType

string

true

发送类：1表示仅发送短信，2表示仅发送wxSms，3表示仅发送WxBot。多个用英文逗号','分隔

businessType

string

true

业务类型，目前写,RMS,后续可定制

Request-body:

{
  "projectName": "Zaful",
  "projectPlatform": "【Zaful-安卓APP】",
  "businessName": "【Zaful-安卓APP-下单】",
  "behavior": "下单行为",
  "time": "【60分钟】",
  "plat": "下单操作",
  "sendType": "1",
  "businessType": "RMS"
}

Response-example:

{
  "success": true,
  "message": "success",
  "code": "200"
}

Curl-example:

curl -X POST -H 'Content-Type: application/json' -i http://{{host}}:{{port}}/message/push --data '{
  "projectName": "Zaful",
  "projectPlatform": "【Zaful-安卓APP】",
  "businessName": "【Zaful-安卓APP-下单】",
  "behavior": "下单行为",
  "time": "【60分钟】",
  "plat": "下单操作",
  "sendType": "1",
  "businessType": "RMS"
}'
